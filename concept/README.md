- 位
  - 位是信息论、计算机和数字通信中的基本单位，是 binary digit的组合。呈现形式有0/1、true/false、on/off、yes/no等等。
- 字节
  - 8位组成一个字节，为什么是8位组成一个字节呢？
- 字符
  - 汉字和英文字母都可算作是字符
